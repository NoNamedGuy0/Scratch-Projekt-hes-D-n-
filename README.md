# Hungry Dino
Ebben a játékban egy éhes dinót irányítasz, a cél hogy minél több almát megegyél.

## A játék működése
- Az egeret mozgatva a dinó jobbra-balra halad a képernyő alján.
- Ha elkap egy almát, a pontszám nő, és egy hangeffekt hallható.
- Ha az alma leesik a földre, újraindul a képernyő tetejéről. A játék végtelenített.
